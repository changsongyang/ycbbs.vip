配置好了 `Git`，我们就可以来学习如何使用 `Git` 进行日常的开发

使用 `Git` 日常开发的流程一般如下

1、  创建或克隆 `Git` 资源作为工作目录
2、  在克隆的资源上添加或修改文件
3、  如果其他人修改了，我们可以更新资源
4、  在提交前查看修改
5、  提交修改到本地
6、  提交修改到远程
7、  在修改完成后，如果发现错误，可以撤回提交并再次修改并提交
